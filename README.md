# <h1 align="center"> MYeye</h1>
<h3 align="center">MYeye is a platform dedicated to providing a seamless shopping experience for eyewear products.</h3>
<br/>

## Features

- Home Page
- Product Listing Page
- Filters
- Search
- Cart Management
- Wishlist Management
- User Profile Page
- Address Management
- Checkout Page
- Payment gateway integration
- Authentication
- Responsive
- Loaders and Toasts
